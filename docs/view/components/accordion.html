Basic

You can create a accordion group using a .panel-group, then using .panel as children.

Use the class .panel-arrow with .panel to display an collapse-arrow in .panel-header.

Margin

You can set a margin when panel are open, using the class .panel-margin.

Popout

You can open then panel like a popout, using the class .panel-popout.
