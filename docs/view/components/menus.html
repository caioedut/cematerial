Basic Style

Default menu styles can be set using the class .menu.

Block

Use the class .menu-block to fill width;

Colors

Feel free to use the classes bg-*-* for background color and text-*-* for text color, defined on Color Palette.

Dropdown Menu

You can put a menu inside the dropdown, and set the position using the rules defined on Dropdown section.

Remember you can cascade the dropdown so submenus using the class .dropdown on li.
